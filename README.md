# professions
Simple menu that you can trigger after character is created
exports.flex_professions:OpenProfessionMenu('intro')

Intro is the first profession menu in the config
You can add as many professions as you want / create new one

Be sure to add the SQL

When you selected a profession or applied for one you will enter a cutscene where you take a vehicle to the setup location for example police to police station
You can even setup rewards after the ride is over
